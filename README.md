Letterwheel is a learning aid for teachers and parents to work with a child on reading and pronouncing simple words. Different patterns of consonants and vowels can be selected by toggling the c/v button. The up and down arrows allows the letters to be changed. 

Letterwheel will present any select pattern of letters. It might or might not be a word. This allows the child to learn and recognize known words from non-words. It also allows pronunciation practice of nonsense words.

Letterwheel can be set to display words from 1 to 5 letters. The learner can be shown particular word families such as -an, where changing the first letter allows words such as ban, can, fan, man, pan, etc.
